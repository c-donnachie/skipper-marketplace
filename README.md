# skipper-marketplace

Marketplace personal de plugins de Claude Code.

## Plugins disponibles

- **[skipper](https://github.com/c-donnachie/skipper)** — 🐧 El capitán de tu documentación. Mantiene docs/architecture, business, decisions (ADR), prds y plans automáticamente.

## Cómo usar

```
/plugin marketplace add c-donnachie/skipper-marketplace
/plugin install skipper@c-donnachie
```
